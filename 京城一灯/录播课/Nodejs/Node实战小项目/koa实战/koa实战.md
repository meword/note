#### 快速搭建 koa2 项目

首先使用npm，全局安装 koa-generator

``` bash
npm install koa-generator -g
```

之后到项目目录使用命令：

``` bash
koa2
```

生成了 koa2 的架构，接着调用：

``` bash
npm install
```

下载相关依赖包，最后使用：

``` bash
npm start
```

启动服务即可。
