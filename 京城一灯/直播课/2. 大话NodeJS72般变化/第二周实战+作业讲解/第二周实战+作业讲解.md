#### 项目结构

```javascript
assets			// 静态资源
config			// 配置文件
controllers		// 路由
models			// 后台请求用的models
middleware		// 中间件
tests			// 测试
views			// 前端的界面
widgets/components	// 组件

web				// 前后端分离的前端文件夹
```









#### 进程

进程有分前台进程和后台进程

前台进程就是输入一个指令，服务必须挂起，不能关闭，关闭了就结束了进程

后台进程就是输入一个指令，服务无需挂起，服务会在后台跑，也看不到
